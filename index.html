My Projects

Stock Market Simulator

A C++ machine‑learning program leveraging NLP to classify forum postings while optimising data structures for large‑scale, real‑time analysis.

Machine‑Code Assembler & Simulator

Engineered a system in C++/Python using NetworkX and Matplotlib to compute and visualise optimal paths in weighted graphs, with dynamic adjustments for sparse graphs.
